# simulator

Robot Simulator based on pybullet and pinocchio

### Installation

    sudo apt install robotpkg-py38-qt5-gepetto-viewer-corba
    sudo apt-get install python-gi-cairo
    sudo apt install robotpkg-py38-example-robot-data
